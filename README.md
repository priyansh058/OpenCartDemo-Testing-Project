# OpenCart Automation Testing Project

## Overview
This project is an automated test suite for the OpenCart e-commerce application using Selenium WebDriver and TestNG.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

##Project Structure
- src/test/java → core logic
- src/test/java → test cases
- testData → test data files

## ✅ Features Covered
- Login 
- Logout
- Register Account
- Forgot Password
- Search Functionality
- Add to Cart

- pom.xml → Maven dependencies
- testng.xml → Test suite configuration
- master.xml → Test suite configuration

## Testing Approach

### Automation Testing
- Implemented using **Page Object Model (POM)**
- Test execution managed via **TestNG**
- Reusable utilities and base classes included

### Manual Testing
- Detailed test cases created for major functionalities
- Bug reports documented for identified issues

---
## ▶️ How to Run the Project

1. Clone the repository:
   https://github.com/priyansh058/OpenCartDemo-Testing-Project
2. Navigate to project folder:
   cd OpenCartDemo-Testing-Project
3. Run tests:
   mvn clean test
